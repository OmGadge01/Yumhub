# 🍴 Yumhub

A simple and interactive recipe finder built with **React + Vite**.\
Search recipes by ingredient, view detailed instructions, and save your
favorites.

------------------------------------------------------------------------

## 🌐 Live Demo

[Live App Here 🚀](https://yumhub-neon.vercel.app/)

------------------------------------------------------------------------


## 🚀 Features

-   🔍 Search recipes by ingredient (e.g., `chicken`, `rice`, `egg`).\
-   📖 View full recipe details (ingredients, instructions, YouTube
    tutorial).\
-   ❤️ Save favorite recipes to local storage for quick access.\
-   🎲 Random recipe generator (for cooking inspiration).\
-   📱 Fully responsive design with **TailwindCSS**.

------------------------------------------------------------------------

## 🗂 Folder Structure

See [`Structure.txt`](./Structure.txt) for full folder layout.

## 🛠 Tech Stack

-   **React (Vite)** → Frontend framework\
-   **TailwindCSS** → Styling\
-   **TheMealDB API** → Recipe data\
-   **LocalStorage** → Favorites persistence

------------------------------------------------------------------------

## 🔧 Setup & Run Locally

1.  Clone the repo:

    ``` bash
    git clone https://github.com/your-username/flavorfinder.git
    cd flavorfinder
    ```

2.  Install dependencies:

    ``` bash
    npm install
    ```

3.  Start development server:

    ``` bash
    npm run dev
    ```

4.  Open in browser:

        http://localhost:5173
